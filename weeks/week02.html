Linear Regression (การถดถอยเชิงเส้น)

จาก "AI คืออะไร" สู่การลากเส้นทำนายค่าด้วยมือ — ก้าวแรกที่จับต้องได้ของ Machine Learning

เป้าหมายสัปดาห์นี้ — เรียนจบแล้วนักศึกษาต้อง:
  1. เห็นภาพรวมว่า AI ⊃ ML ⊃ DL สัมพันธ์กันอย่างไร และ Predictive / Generative / Agentic AI ต่างกันตรงไหน
  2. อธิบายได้ว่า Linear Regression คืออะไร — ความชัน (a) และจุดตัดแกน (b) ในสมการ y = ax + b หมายความว่าอะไร
  3. คำนวณหาสมการเส้นตรงจากข้อมูลด้วยมือด้วยวิธี Least Squares (หา x̄, ȳ → ตาราง → a, b) แล้วนำไปทำนายค่าได้
  4. บอกประเภทของ regression (Linear, Polynomial, Multiple, Logistic) และยกตัวอย่างการใช้งานจริงได้

ส่วนที่ 1 — แนวคิด: AI สี่ยุค แนวคิด

คำถามเปิดคาบ: "ChatGPT ฉลาดเพราะอะไร — มีคนเขียนกฎคำตอบใส่ไว้ หรือมันเรียนรู้เอง?"

  • ยุคที่ 1: Symbolic AI / กฎที่มนุษย์เขียน (1950s–1980s)
    ความฉลาด = กฎ if-then ที่ผู้เชี่ยวชาญเขียนใส่เครื่อง (expert systems) เก่งในโลกแคบๆ เช่นเล่นหมากรุก แต่พังเมื่อเจอโลกจริงเพราะเขียนกฎให้ครอบคลุมทุกกรณีไม่ไหว
  • ยุคที่ 2: Machine Learning (1990s–2000s)
    เปลี่ยนวิธีคิด: ไม่เขียนกฎเอง แต่ให้เครื่อง "เดากฎ" จากตัวอย่างข้อมูล — regression, decision tree, SVM ใช้ได้จริงในวงกว้าง แต่มนุษย์ยังต้องเลือก feature ให้
  • ยุคที่ 3: Deep Learning (2012–2017)
    neural network หลายชั้น หา feature เองได้จากข้อมูลดิบ จุดระเบิดคือ AlexNet (2012) ชนะการแข่งจำแนกรูปภาพแบบขาดลอย ตามด้วย AlphaGo (2016) — แต่ด้านภาษายังติดขัด
  • ยุคที่ 4: Transformer & LLM (2017–ปัจจุบัน)
    งานวิจัย "Attention Is All You Need" (2017) ให้กำเนิดสถาปัตยกรรม Transformer → GPT, ChatGPT (2022), Claude, Gemini

3 ประเภท AI ที่ได้ยินบ่อยที่สุดตอนนี้

Predictive → Generative → Agentic คือลำดับการพัฒนาของ AI — ทั้งหมดต่อยอดจากแก่นเดียวที่เราเรียนวันนี้คือ "ทำนายตัวถัดไป" (กดที่กล่องเพื่อไฮไลท์)

Predictive AI — AI ที่ "ทำนาย/พยากรณ์" จากข้อมูล
ดูข้อมูลในอดีตแล้ว ทำนายค่าหรือหมวดหมู่ ออกมา เช่น พยากรณ์ยอดขายเดือนหน้า, คัดอีเมลว่าเป็นสแปมไหม, ทำนายว่าลูกค้าจะเลิกใช้บริการหรือเปล่า
ผลลัพธ์เป็น "คำตอบสำเร็จรูป" (ตัวเลข/ป้ายกำกับ) ไม่ได้สร้างเนื้อหาใหม่ — เป็น AI ยุค Machine Learning / Deep Learning ที่ใช้กันมานาน
Generative AI — AI ที่ "สร้างเนื้อหาใหม่"
ไม่ได้แค่ทายคำตอบเดียว แต่ สร้างสิ่งใหม่ทั้งชิ้น — ข้อความ รูปภาพ เสียง วิดีโอ โค้ด เช่น ChatGPT แต่งบทความ, AI วาดรูปจากคำบรรยาย
วิธีการ: ทำนายชิ้นถัดไปต่อๆ กันเป็นลูกโซ่ จนได้ผลงานใหม่ที่ไม่เคยมีมาก่อน
Agentic AI — AI ที่ "ลงมือทำเอง" ไม่ใช่แค่ตอบ
เอา Generative AI มาต่อให้ วางแผน → เรียกเครื่องมือ (ค้นเว็บ เขียนไฟล์ สั่งโปรแกรม) → ดูผล → ทำขั้นต่อไปเอง วนซ้ำจนงานเสร็จ
ตัวอย่าง: บอกว่า "จองตั๋วถูกสุดให้หน่อย" แล้วมันไปค้นราคา เทียบ กดจองให้เอง — แทนที่จะแค่บอกวิธีจอง

ส่วนที่ 2 — เริ่มรู้จัก Regression แนวคิด

Regression — การถดถอย

เทคนิคใน Machine Learning (อยู่ในวง ML ที่เพิ่งดูไป) ที่ "ลากเส้นหรือเส้นโค้งให้เข้ากับข้อมูล" เพื่อ ทำนายค่าตัวเลขต่อเนื่อง เช่น ราคา อุณหภูมิ ยอดขาย จากปัจจัยนำเข้า
ต่างจาก classification ที่ทายเป็น "หมวดหมู่" — regression ทายเป็น "ตัวเลข"

มีกี่ประเภท?

Linear

เส้นตรงเส้นเดียว y = ax + b — ใช้เมื่อข้อมูลมีแนวโน้มเชิงเส้น

Polynomial

เส้นโค้ง รองรับความสัมพันธ์ที่ไม่เป็นเส้นตรง

Multiple

หลายปัจจัย (x₁, x₂, x₃ …) ทำนาย y เดียว เช่น ราคาบ้านจากพื้นที่ + ทำเล + อายุ

Logistic

เส้นโค้ง S ใช้ "จำแนกหมวด" (ใช่/ไม่ใช่) — ชื่อมี regression แต่ทำ classification

เอาไว้ทำอะไรบ้าง?

  • พยากรณ์ราคา — บ้าน รถ หุ้น ที่ดิน
  • คาดการณ์ยอดขาย / ดีมานด์ในอนาคต
  • ประมาณการอุณหภูมิ / การใช้พลังงาน
  • หาความสัมพันธ์ของปัจจัย (เช่น งบโฆษณา → ยอดขาย)
  • ประเมินความเสี่ยง / แนวโน้มทางการแพทย์และการเงิน

ส่วนที่ 3 — Linear Regression ลงลึก ลงมือ

Linear Regression คือการหา เส้นตรงเส้นเดียว ที่ "แนบ" กับข้อมูลมากที่สุด แล้วใช้เส้นนั้นทำนายค่าใหม่ หัวใจอยู่ที่สมการเส้นตรง:

y=a·x+b
a = ความชัน (slope) — x เพิ่มขึ้น 1 หน่วย แล้ว y เปลี่ยนไปเท่าไร
b = จุดตัดแกน y (intercept) — ค่า y ตอนที่ x = 0

ลองปรับเส้นเอง — แล้วดู "ความคลาดเคลื่อน" เปลี่ยน

เส้นประแดงคือระยะที่เส้นพลาดจากจุดจริง คอมพิวเตอร์เลือกเส้นที่ทำให้ ผลรวมของระยะนั้นยกกำลังสอง (SSE) น้อยที่สุด — เรียกว่า Least Squares

ส่วนที่ 4 — ตัวอย่างโจทย์จริง: หาสมการจาก 10 จุด ลงมือ

โจทย์: ร้านน้ำหวานจดข้อมูล 10 วัน ว่าวันที่อุณหภูมิต่างกัน ขายน้ำได้กี่แก้ว เราจะหา "สมการเส้นตรง" เพื่อ ทำนายยอดขายจากอุณหภูมิ

แตะ/ลากบนกราฟเพื่ออ่านค่า (x, y)
  1. หาค่าเฉลี่ยของทั้งสองชุด
    x̄ (อ่านว่า "เอ็กซ์-บาร์") คือ ค่าเฉลี่ยของ x และ ȳ ("วาย-บาร์") คือ ค่าเฉลี่ยของ y
    สูตร: ค่าเฉลี่ย = ผลรวมทั้งหมด ÷ จำนวนข้อมูล  →  x̄ = Σx / n  (สัญลักษณ์ Σ "ซิกมา" แปลว่า "ผลรวมของทั้งหมด" ส่วน n คือจำนวนจุด = 10)
    Σx = 28+29+…+37 = 325 → x̄ = 325 / 10 = 32.5
    Σy = 45+50+…+102 = 735 → ȳ = 735 / 10 = 73.5
  2. สร้างตารางหา "ส่วนต่างจากค่าเฉลี่ย" แล้วคูณ/ยกกำลังสอง

    แต่ละคอลัมน์คืออะไร และทำไมต้องหา?

    • x − x̄ และ y − ȳ — เรียกว่า ส่วนเบี่ยงเบนจากค่าเฉลี่ย (deviation from the mean) บอกว่าจุดนั้นอยู่ เหนือหรือใต้ ค่าเฉลี่ยเท่าไร — เราใช้ค่าเฉลี่ยเป็นจุดอ้างอิงเพราะมันคือ "ศูนย์กลาง" ของข้อมูล
    • (x − x̄)(y − ȳ) — ผลคูณส่วนเบี่ยงเบน พอรวมกัน (Σ) จะเป็นตัวตั้งของ ความแปรปรวนร่วม (covariance) ที่วัดว่า x กับ y "ไปทางเดียวกันไหม": ถ้า x มากกว่าเฉลี่ยแล้ว y มักมากกว่าเฉลี่ยด้วย ผลคูณจะเป็นบวก → ความสัมพันธ์เชิงบวก (เส้นชันขึ้น)
    • (x − x̄)² — ส่วนเบี่ยงเบนของ x ยกกำลังสอง รวมกันคือตัวตั้งของ ความแปรปรวนของ x (variance) ที่วัดว่า x กระจายมากแค่ไหน — ยกกำลังสองเพื่อให้เป็นบวกเสมอ ค่าบวกลบจะได้ไม่หักล้างกัน

    ทำไมเอามาหารกัน? สูตร a = Σ(x−x̄)(y−ȳ) ÷ Σ(x−x̄)² ไม่ได้สุ่มมา แต่เป็นคำตอบที่พิสูจน์ได้จากวิธี Least Squares (เลือกเส้นที่ทำให้ค่าคลาดเคลื่อนยกกำลังสองน้อยที่สุด) อ่านง่ายๆ ว่า ความชัน = (x กับ y แปรไปด้วยกันแค่ไหน) ÷ (x แปรปรวนแค่ไหน) — เป็นสูตรมาตรฐานที่ใช้กันทั่วโลก

  4. คำนวณจุดตัดแกน b = ȳ − a·x̄ = 73.5 − (6.38 × 32.5) ≈ 73.5 − 207.4 ≈ −133.9

  6. นำไปใช้ทำนาย — ลองใส่อุณหภูมิเองแล้วดูยอดขายที่โมเดลทำนาย
