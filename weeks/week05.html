K-means: จับกลุ่มข้อมูลเอง (โจทย์ที่ "ไม่มีเฉลย")

สัปดาห์ที่แล้ว KNN มี "ป้ายคำตอบ" ให้ดูเพื่อนบ้าน — แต่ถ้าไม่มีใครติดป้ายไว้เลยล่ะ? K-means ให้เครื่อง ค้นพบกลุ่มเอง จากความใกล้ในปริภูมิ นี่คือประตูสู่ Unsupervised Learning

เป้าหมายสัปดาห์นี้ — เรียนจบแล้วนักศึกษาต้อง:
  1. แยกได้ว่า Supervised (มีเฉลย เช่น KNN, Classification) ต่างจาก Unsupervised (ไม่มีเฉลย เช่น Clustering) อย่างไร
  2. อธิบายวงจร Lloyd's algorithm ของ K-means ได้ 2 ขั้นวน: ① จับจุดเข้าเซนทรอยด์ที่ใกล้สุด → ② ขยับเซนทรอยด์ไปที่ค่าเฉลี่ยของสมาชิก → ซ้ำจนนิ่ง
  3. คำนวณด้วยมือ 1–2 รอบได้ และเข้าใจว่า inertia (WCSS) ลดลงทุกรอบจนลู่เข้า
  4. รู้จุดอ่อน: ต้องเลือก K เอง (elbow method), ไวต่อ จุดเริ่ม (k-means++), และชอบกลุ่มทรงกลม
  5. เชื่อมโยงได้ว่า clustering บนเวกเตอร์ embedding คือหัวใจของการจัดกลุ่มความหมาย, vector quantization และการสร้าง "โค้ดบุ๊ก" ใน AI

ส่วนที่ 1 — แนวคิด: "ไม่มีใครบอกว่าถูก แล้วจะรู้กลุ่มได้ไง?" แนวคิด

ลองนึกภาพ: เทข้อมูลลูกค้า 500 คน (อายุ, ยอดใช้จ่าย) ลงกระดาษ — ไม่มี ใครติดป้ายว่าใครเป็น "กลุ่มไหน" แต่พอมองด้วยตา เราก็ เห็นก้อน ของจุดที่เกาะกันเป็นกระจุกอยู่ดี สมองเราจับกลุ่มเองอัตโนมัติ — K-means คือการสอนให้เครื่องทำแบบนั้น

ส่วนที่ 3 — K-means ทำงานทีละขั้น (มโนภาพการวน) ทีละขั้น

ในเวทีด้านบน กดปุ่ม ⏭ ทีละขั้น ค้าง ๆ แล้วจับคู่กับลำดับนี้ จะเห็นชัดว่าแต่ละกดคือครึ่งรอบ (สลับ ①/②):

  1. เริ่ม (Initialize): สุ่มวางเซนทรอยด์ K อันลงในสนาม — ยังไม่มีจุดไหนมีสี (ยังไม่สังกัดกลุ่ม)
  2. ① Assign: แต่ละจุดมองหาเซนทรอยด์ที่ใกล้ตัวที่สุด แล้วเปลี่ยนสีตามกลุ่มนั้น (เกิด "อาณาเขต" ให้เห็น)
  3. ② Update: เซนทรอยด์แต่ละอันเลื่อนไปที่ค่าเฉลี่ยตำแหน่งของสมาชิกในกลุ่มมัน (✕ ขยับเข้าใจกลางก้อน)
  4. วนซ้ำ: พอเซนทรอยด์ขยับ จุดบางจุดก็มีเจ้าของใหม่ → กลับไปทำ ① แล้ว ② อีก inertia ลดลงทุกครั้ง
  5. ลู่เข้า (Converged): เมื่อทำ ① แล้วไม่มีจุดไหนเปลี่ยนกลุ่ม และ ② เซนทรอยด์แทบไม่ขยับ → หยุด ป้ายขึ้น "นิ่งแล้ว!"
  6. ตีความ: กลุ่มที่ได้คือ "โครงสร้างที่ K-means ค้นพบ" — เอาไปตั้งชื่อ/วิเคราะห์ต่อได้ (เช่น "กลุ่มลูกค้าใช้จ่ายสูง")

ห้องแล็บ K-means — วางจุดเองไม่กี่จุด แล้วดู "เลขจริง" ของทุกรอบ Assign / Update

คลิกวางจุด สัก 4–6 จุด (พิกัดล็อกที่ .5 ให้เลขสวย) — เซนทรอยด์ ✕ ลากวางเองได้ แล้วกดปุ่มเดินทีละครึ่งรอบ: เครื่องจะกางตารางระยะทางทุกคู่ (สูตร √ พร้อมเลขข้างใน) ไฮไลต์ตัวที่ชนะ → แล้วโชว์การเฉลี่ยพิกัดใหม่แบบเห็นผลบวกจริง วนจนลู่เข้า

ข้อ 2 (ทำในคาบ) — K-means ≠ KNN

เพื่อนสับสนว่า "K ในสองอัลกอริทึมนี้เหมือนกัน" จงอธิบายให้ชัดว่า K ใน KNN (เรื่อง KNN) กับ K ใน K-means คืออะไร ต่างกันอย่างไร และอันไหน supervised / unsupervised

ดูเฉลย KNN: K = จำนวนเพื่อนบ้านที่เอามาโหวตตอนทำนายคลาสของจุดใหม่ — เป็น supervised (ต้องมีป้ายคำตอบ)
K-means: K = จำนวนกลุ่มที่ต้องการแบ่ง — เป็น unsupervised (ไม่มีป้าย ค้นกลุ่มเอง)
เหมือนกันแค่แนวคิด "วัดระยะ = ความคล้าย" แต่คนละงานคนละจังหวะการใช้
ข้อ 3 (การบ้าน) — เลือก K และรับมือจุดอ่อน

(ก) อธิบาย elbow method ว่าใช้ inertia เลือก K อย่างไร ทำไมเราไม่เลือก K ที่ inertia ต่ำสุด (K = จำนวนจุด)
(ข) ยกตัวอย่างข้อมูลรูปทรงที่ K-means จะจับกลุ่มพลาด และเสนออัลกอริทึมทางเลือก
(ค) k-means++ ช่วยแก้ปัญหาอะไร และแก้ด้วยหลักการใด (การบ้าน — ไม่มีเฉลย)

ข้อ 4 (การบ้าน) — โยงสู่ AI จริง

ระบบหนึ่งแปลงรีวิวสินค้า 10,000 ชิ้นเป็นเวกเตอร์ embedding แล้วใช้ K-means จับกลุ่ม จงอธิบายว่า "จุด" และ "ระยะทาง" ในที่นี้คืออะไร, ผลลัพธ์ K กลุ่มเอาไปทำอะไรได้ (เช่นสรุปหัวข้อที่ลูกค้าพูดถึง), และทำไมคุณภาพ embedding จึงสำคัญต่อผลการจับกลุ่ม (การบ้าน — ไม่มีเฉลย)

ส่วนที่ 6 — ขั้นบันไดของสัปดาห์นี้ ขั้นบันได

สัปดาห์นี้พาเราเข้าใกล้ LLM อย่างไร:
K-means เปิดโลก Unsupervised Learning — "เรียนรู้โครงสร้างจากข้อมูลดิบที่ไม่มีเฉลย" ซึ่งเป็นจิตวิญญาณเดียวกับที่ LLM ถูก pretrain บนข้อความมหาศาลที่ไม่มีใครติดป้าย
ที่ตรงกว่านั้น: เมื่อคำ/เอกสารกลายเป็นเวกเตอร์ embedding การ clustering ก็คือการจัดกลุ่ม "ความหมายที่คล้ายกัน" เข้าด้วยกัน — ใช้สรุปหัวข้อ (topic discovery), ลดข้อมูลซ้ำ, และจัดระเบียบคลังความรู้ก่อนป้อน RAG · แนวคิด vector quantization (แทนเวกเตอร์ด้วย "เซนทรอยด์/โค้ดบุ๊ก" ที่ใกล้สุด) เป็นหัวใจของโมเดลอย่าง VQ-VAE และตัวบีบอัด token ของโมเดลภาพ/เสียง — ทั้งหมดคือ K-means ในเสื้อคลุมใหม่
สัปดาห์หน้า: เราจะกลับสู่โลก supervised อีกครั้งด้วย Neural Network — สมองจำลองที่ เรียนรู้ เส้นแบ่งเขต/ความน่าจะเป็นเองได้ (รวมถึง softmax ที่ LLM ใช้เลือกคำ) — บล็อกก่อสร้างของ AI ยุคใหม่ทั้งหมด
