ปัญญาประดิษฐ์ (Artificial Intelligence)

ภาคเรียนที่ 1/2569 — เส้นทางจากการทำนายตัวเลข สู่การเข้าใจว่า LLM ทำงานอย่างไร

⚡ สิ่งที่ LLM ทำตลอดเวลา — ทำนายคำถัดไป (วิชานี้จะพาไปดูข้างในว่าทำได้อย่างไร): ▌
แกนของวิชานี้: AI สมัยใหม่คือ "เครื่องทำนายที่เรียนรู้จากข้อมูล"
เราจะเริ่มจากเครื่องทำนายที่เล็กที่สุด แล้วต่อบันไดขึ้นทีละขั้น — ทำนายตัวเลข → ทำนายหมวดหมู่ → ทำนายจากภาพ → ทำนายคำถัดไป — จนสุดท้ายนักศึกษาตอบได้ด้วยตัวเองว่า ChatGPT/Claude ข้างในทำงานอย่างไร

รูปแบบของทุกสัปดาห์

1. แนวคิด เนื้อหาหลักพร้อมภาพประกอบ   2. ทดลองเล่น demo แบบโต้ตอบบนหน้าเว็บ   3. แบบฝึกหัด ทำในคาบ + การบ้าน   4. ขั้นบันได สรุปว่าสัปดาห์นี้พาเข้าใกล้ LLM อย่างไร

🪜 เส้นทางการเรียนรู้ทั้งหมด

  1. AI คืออะไร มาจากไหน — และโมเดลภาษาที่เล็กที่สุดในโลก ประวัติ AI 4 ยุค + ลองสร้าง language model จิ๋วด้วยมือ
    เห็นภาพรวมว่า LLM คือปลายทางของเส้นทาง 70 ปี และแก่นของมันคือ "การทำนายตัวถัดไป"
  2. Linear Regression — การถดถอยเชิงเส้น (ฉบับลงมือทำ) ลากเส้นทำนายค่าด้วย Least Squares — เดโมปรับเส้นสด, ตัวอย่างจริง 2 โจทย์, และสร้างข้อมูลเอง
    หัวใจ ML: หา weight (a, b) ที่ทำให้ความคลาดเคลื่อนน้อยที่สุด — แนวคิดเดียวกับที่ LLM ใช้เทรน เพียงแต่ LLM มี weight เป็นแสนล้านตัว
  3. เมื่อข้อมูลไม่เป็นเส้นตรง — Polynomial Regression มุมมอง 3 มิติ "ยกมิติ z=x²" + เวทีเล่นดัด degree สด พร้อมกราฟตัว U ของ overfitting วาดตัวเอง
    บทเรียนสำคัญของ ML: ยิ่งโมเดลซับซ้อน (degree สูง) ยิ่ง fit ข้อมูลฝึกเป๊ะ แต่ทำนายข้อมูลใหม่แย่ลง — ที่มาของการแยกชุดฝึก/ทดสอบ
  4. จำแนกด้วยเพื่อนบ้านใกล้ที่สุด — K-Nearest Neighbors (KNN) จัดประเภทของใหม่จากเสียงข้างมากของเพื่อนบ้าน k จุดที่ใกล้ที่สุด — วัดระยะ, เลือก k, โหวต
    บทเรียนสำคัญของ AI: "ความหมาย = ความใกล้ในปริภูมิ" — KNN บนเวกเตอร์ embedding คือหัวใจของ semantic search และ RAG ที่ LLM ใช้ค้นความรู้
  5. จับกลุ่มข้อมูลเอง — K-means Clustering Unsupervised: เวทีอนิเมชัน assign/update จนลู่เข้า + Elbow method รันสด + บีบสีรูปภาพเหลือ K สี
    โลกของ "เรียนรู้จากข้อมูลที่ไม่มีเฉลย" แบบเดียวกับที่ LLM ถูก pretrain — และ clustering บน embedding คือหัวใจของ vector quantization / การจัดกลุ่มความหมาย
  6. Decision Tree & Random Forest — เครื่องที่ "ถามทีละคำถาม" ต้นไม้ if/else ที่อ่านออกได้ + เวทีแบ่งพื้นที่เห็นเส้นขั้นบันได + เลือกจุดตัดด้วย Gini ทีละสเต็ป + ป่าสุ่มโหวตรวมพลัง
    โมเดลที่ "อธิบายได้" และครองข้อมูลตาราง (tabular/Kaggle) — ตรงข้ามกับกล่องดำอย่าง Neural Network ที่กำลังจะเรียน
  7. SVM — เส้นแบ่งที่กว้างที่สุด (และเวทมนตร์ Kernel) หาเส้นที่ margin กว้างสุด + จุด support vectors + ปุ่ม C ปรับความยืดหยุ่น + ยก 2 มิติขึ้น 3 มิติให้ระนาบแบ่งได้
    แชมป์ยุคก่อน deep learning — เข้าใจ margin/kernel แล้วจะซึ้งว่าทำไม Neural Network ถึงมาแทนในเวลาต่อมา
  8. Neural Network — สมองจำลอง จาก Perceptron สู่ MLP ไทม์ไลน์ประวัติ 80 ปี + นิวรอนใส่ค่าเอง + MLP Playground เห็นค่าวิ่งทุกโหนด + แก้ปริศนา XOR + softmax
    ได้ "หน่วยก่อสร้าง" ที่ LLM ใช้จริง — ข้างใน Transformer คือ MLP ซ้อนกัน และ softmax ชั้นสุดท้ายคือวิธีที่ LLM เลือก "คำถัดไป"
  9. Backpropagation — เครื่องจักรเรียนรู้ได้อย่างไร (ฉบับตามได้ทุกตัวเลข) ลูกบอลกลิ้งลงเขา gradient descent + chain rule + ห้องแล็บ backprop เห็นเลขจริงทุกขั้น เทรนวนดู loss ลด
    นี่คืออัลกอริทึมเดียวกันเป๊ะที่ใช้เทรน GPT — ไม่มีเวทมนตร์อื่น
  10. CNN — สอนเครื่องให้มองเห็น วาดภาพเอง + convolution ไถลทีละช่องเห็นเลขคูณ-บวก + pooling + เดโมจำแนก O/X ทั้งไปป์ไลน์
    บทเรียนสำคัญ: สถาปัตยกรรมที่เหมาะกับชนิดข้อมูลชนะเสมอ — แล้วข้อความล่ะ ควรใช้สถาปัตยกรรมแบบไหน?
  11. เทรนจริงให้เป็น — Loss, Overfitting, Train/Test Split ปฏิบัติการเทรนโมเดลใน Colab
    เข้าใจว่าทำไม LLM ต้องใช้ข้อมูล "ทั้งอินเทอร์เน็ต" — เพราะโมเดลใหญ่ overfit ง่าย
  12. สอบกลางภาค + ทบทวนครึ่งทาง
    ครึ่งแรกจบ: เรามีเครื่องทำนายที่เรียนรู้ได้แล้ว — ครึ่งหลังคือทำให้มัน "เข้าใจภาษา"
  13. ทำนายคำถัดไป — Language Model, RNN/LSTM และทำไมมันไม่พอ จาก n-gram (เรื่อง AI คืออะไร) สู่ RNN — และปัญหาความจำสั้น
    เจอกำแพงที่ทำให้ AI ภาษา "ติดหล่ม" อยู่หลายปี — กำแพงที่ Transformer จะทลายในสัปดาห์หน้า
  14. Attention & Transformer — หัวใจของ AI ยุคปัจจุบัน attention คือการ "เลือกมองคำที่เกี่ยวข้อง", โครงสร้าง Transformer ทีละบล็อก
    ⭐ ขั้นสำคัญที่สุดของเทอม — สถาปัตยกรรมตัวจริงข้างใน GPT, Claude, Gemini
  15. LLM ทำงานอย่างไร — Tokenization, Pretraining, Fine-tuning, RLHF ต่อจิ๊กซอว์ทุกชิ้นที่ผ่านมาเข้าด้วยกัน
    🏁 ถึงยอดบันได: นักศึกษาอธิบายได้ครบวงจรว่า ChatGPT ทำงานอย่างไร และทำไมมันถึง hallucinate
  16. ใช้ LLM อย่างคนที่รู้ข้างใน — Prompting, RAG, Agents เขียน prompt อย่างมีหลักการ, ต่อ LLM กับข้อมูลภายนอก
    เปลี่ยนจาก "เข้าใจ" เป็น "ใช้งานจริง" — ทักษะที่ตลาดงานต้องการที่สุดตอนนี้
  17. AI สร้างภาพและเสียงได้อย่างไร — Diffusion & Multimodal หลักการเดียวกัน (ทำนาย) แต่ทำนาย "noise ที่ต้องลบออก"
    เห็นว่าแก่นความรู้ทั้งเทอมใช้อธิบาย AI ชนิดอื่นได้หมด ไม่ใช่แค่ภาษา
  18. ข้อจำกัด จริยธรรม และอนาคต hallucination, bias, ผลกระทบต่ออาชีพ, การใช้อย่างรับผิดชอบ
    เพราะเข้าใจข้างในแล้ว จึงวิจารณ์ได้อย่างมีหลักการ ไม่ใช่กลัวหรือเชื่อแบบงมงาย
  19. นำเสนอโปรเจกต์ปลายภาค
    โจทย์เปิด: สร้างหรือวิเคราะห์ระบบ AI หนึ่งอย่าง แล้วอธิบาย "ข้างใน" ให้เพื่อนเข้าใจ
