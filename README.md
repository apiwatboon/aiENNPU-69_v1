# สื่อการสอนวิชาปัญญาประดิษฐ์ — ภาคเรียนที่ 1/2569

เว็บไซต์สื่อการสอนแบบ static HTML (ไม่ต้อง build, ไม่ต้องติดตั้งอะไร) —
เปิด `index.html` ในเบราว์เซอร์ได้เลย หรือ deploy ขึ้น GitHub Pages ได้ทันที

## แนวคิดของหลักสูตร

สอนแบบ **"บันไดเป็นขั้น"** — แกนเรื่องคือ *"AI สมัยใหม่ = เครื่องทำนายที่เรียนรู้จากข้อมูล"*
ทุกสัปดาห์ต่อยอดกันจนสัปดาห์ที่ 13 นักศึกษาอธิบายได้ครบวงจรว่า LLM ทำงานอย่างไร

ทุกสัปดาห์มีโครงเดียวกัน 4 ส่วน:

1. **แนวคิด** — เนื้อหา เปิดด้วยคำถามชวนคิดเสมอ
2. **ทดลองเล่น** — demo โต้ตอบบนหน้าเว็บ (JS ในหน้า หรือฝังเครื่องมือภายนอก)
3. **แบบฝึกหัด** — ทำในคาบ (มีเฉลยพับเก็บใน `<details>`) + การบ้าน
4. **ขั้นบันได** — สรุปว่าสัปดาห์นี้พาเข้าใกล้ LLM อย่างไร และโยงไปสัปดาห์หน้า

## โครงสร้างไฟล์

```
index.html          หน้าหลัก: บันได 17 สัปดาห์ + การวัดผล
assets/style.css    สไตล์กลางทุกหน้า (รวม CSS แอนิเมชัน)
assets/anim.js      แอนิเมชันร่วม: reveal เมื่อเลื่อนถึง, typeText, sleep
weeks/week01.html   AI คืออะไร (ตัวอย่างสมบูรณ์ มี bigram demo)
weeks/_template.html  เทมเพลตสำหรับสร้างสัปดาห์ใหม่
```

นอกจากโครง 4 ส่วน ทุกสัปดาห์ที่มีเนื้อหาใหม่ยังมี:

- **เข้าใจทีละขั้นแบบโต้ตอบ** (`ol.steps` + `window.stepScenes`) — กด ▶ เล่นทีละขั้นหรือคลิกขั้นใดก็ได้
  มี "เวทีภาพประกอบ" เปลี่ยนแอนิเมชันตามขั้น (เฟรมเวิร์กอยู่ใน anim.js, ฉากของแต่ละหน้าอยู่ใน script ท้ายไฟล์)
- **กล่องแหล่งเรียนรู้** (`<div class="resources">`) — ลิงก์คอร์ส/วิดีโอระดับโลกที่คัดให้ตรงหัวข้อ (3Blue1Brown, Karpathy, MIT, Stanford ฯลฯ)
- **อนิเมชันเล่าเรื่อง** — ปุ่ม ▶ ใน demo ที่เล่นให้ดูเอง เช่น ไทม์ไลน์ 4 ยุค (ส.1), gradient descent (ส.2–3), เส้นตรง 100 เส้น vs XOR (ส.4), kernel กวาดภาพ (ส.7), denoise (ส.13)

## วิธีเพิ่มสัปดาห์ใหม่

1. คัดลอก `weeks/_template.html` → `weeks/weekNN.html`
2. เติมเนื้อหาตามโครง 4 ส่วน + กล่องเจาะลึก + กล่องแหล่งเรียนรู้
3. แก้ลิงก์ในหน้า index และปุ่ม nav ของสัปดาห์ก่อนหน้า/ถัดไป
